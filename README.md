# Rovr

A Rust-based project for video processing and manipulation.

## Prerequisites

- Rust (latest stable version)
- FFmpeg (version 7.1.1 or compatible)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/EthnTuttle/rovr.git
cd rovr
```

2. Build the project:
```bash
cargo build
```
